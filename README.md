1.  Download the latest `KeyVault.app.zip` from the [Releases](https://github.com/hjm79/keyvault-xcode/releases) page.
2.  Unzip the file and move `KeyVault.app` to your `Applications` folder.

4.  Open KeyVault and enjoy!
